# Prerequisites

* Install [Node.js LTS](https://nodejs.org/en/download/)
* Install [Oracle JDK v1.8](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
* Install chimp:
  ```bash
  npm install -g chimp
  ```
